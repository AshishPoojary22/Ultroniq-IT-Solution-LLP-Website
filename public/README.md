# Public Assets

Place your logo file here with the filename `logo.png`.

The logo should be:
- A PNG file named `logo.png`
- Recommended size: 200x200px or larger (square aspect ratio)
- The logo will be displayed in:
  - Navbar (top left)
  - Footer (company info section)

The logo will automatically be referenced from the root path `/logo.png` in the application.
